# Oceancode Admin Template

# install
```shell
pnpm install

# 使用本地mock接口
pnpm dev:mock
```

## commit 提交格式

type(scope?): subject 换行 body 换行 footer
示例
```
chore: run tests on travis ci
 
feat(blog): add comment section
```
